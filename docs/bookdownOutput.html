1 BIOL8700_setup_2022

The goal of BIOL8700_setup_2022 is to setup and generate documents using RStudio, RMarkdown and github

1.1 Tools

This is a sample book written in Markdown and includes embedded R code. This combination of programming languages is included in the RMarkdown package. We use this and a variation of other RMarkdown packages to make the most of reproducible reporting in R.

1.2 Data importing

There are scripts and code included within this repository to import data from csv and excel files from local and online locations.
